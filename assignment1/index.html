Hi there!

My name is Daniel Borg! As described in the heading, I'm a Maltese born software developer, who's been programming for 13 years now, and it's only slightly broken my mind!
I've done work in Unity, Unreal and Godot engine before, but my knowledge base is not limited to game development. I've worked in multiple teams doing front end and back end web development, database management and delved into the deepest pits of Agile development, much to my own cost.
This is my very simple portfolio site, cobbled together from basic css and html. Maybe one day I'll use a framework if I could be bothered.
